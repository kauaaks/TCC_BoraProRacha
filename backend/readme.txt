caso não possua o node no pc, instale pelo site oficial: https://nodejs.org/pt?utm_source=chatgpt.com

instalar o node na pasta:
git bash na pasta do projeto, logo após:
npm install
depois use node -v e npm -v pra testar caso o npm e node n instalem. (tem que ser na pasta mãe do projeto)

pra rodar: npm run dev(utiliza o nodemon) ou npm start (n usa o nodemon)
